Start executing the script.

Memory warning

If you do not consume the values from on_event, they will remain stored forever. You have been warned.

Errors

The first event sent to on_event is a ExecutionEvent::Starting, which informs the caller of whether the execution could start. Possible reasons that would prevent execution are: - RunningError:AlreadyRunning if the script is already running; - a compilation error if the script was incorrect.
